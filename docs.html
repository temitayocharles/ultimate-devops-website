📘 Documentation

This guide covers setup, usage, and common tasks.

🚀 Getting Started

git clone https://github.com/temitayocharles/ultimate-devops
cd ultimate-devops
./devops-env.sh init
./devops-env.sh run

🗃️ Persistent Volume

Data is mounted to /data and persists across sessions.

🔐 Credentials

No secrets are baked into the container. Use vault login or AWS profiles for auth.

🧼 Reset Everything

docker volume rm devops-volume
docker rmi temitayocharles/ultimate-devops
